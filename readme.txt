= How to configure

Put sbts.bat in PATH or in a location accessible from PATH
Change SCALA_SCRIPTS_HOME from sbts.bat to point to the current folder.

Add library dependencies in plugins.sbt
Add scripting commands in Build.scala

= How to execute
`sbts` - enter in console mode where you can use the defined commands
`sbts backup z:\1-personal-backups-auto\2015-04-27--archived--synchting-s4\` - to execute the backup command.
